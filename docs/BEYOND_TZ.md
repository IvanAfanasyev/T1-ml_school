# Достижения сверх ТЗ

## 1. Четыре провайдера

Подключены:

- Т1 Облако;
- Cloud.ru;
- Selectel;
- VK Cloud.

Минимум в ТЗ — 3 провайдера.

## 2. Диалоговый агент

Агент понимает свободный текст, выявляет недостающие параметры и задаёт уточняющие вопросы.

## 3. Инфраструктурные связки

Система умеет подбирать не только один сервис, но и связку:

```text
Compute + Managed Database + Object Storage
```

## 4. Hybrid retrieval

Используется комбинация:

```text
BM25 + semantic embeddings
```

## 5. Многофакторное ранжирование

Учитываются:

- регион;
- бюджет;
- 152-ФЗ;
- технологии;
- компонент;
- use case;
- тарифные позиции.

## 6. Объяснимость

В ответе показываются совпавшие критерии и параметры для проверки.

## 7. Production-ready схема

Есть:

- FastAPI backend;
- Docker Compose;
- Nginx;
- healthcheck;
- отдельный frontend;
- отдельный algorithm layer.

## 8. Golden Dataset и LLM Judge

Используются автоматические метрики и LLM-as-a-judge.

## 9. C4-документация

Подготовлены C4-диаграммы и data flow.
