CloudMatch

Маркетплейс облачных сервисов с диалоговым подбором

Проект помогает выбрать облачный сервис под задачу пользователя: базу данных, Kubernetes, object storage, backend-инфраструктуру и другие сценарии.

Backend работает на FastAPI, использует нормализованные JSON-данные провайдеров, ранжирует сервисы и возвращает готовые карточки для сайта.
