# Frontend

Папка `frontend/` содержит статический интерфейс сайта.

```text
frontend/
  index.html     Разметка вкладок, витрины, агента и страницы о проекте
  styles.css     Визуальная система, карточки, чат, адаптивность
  app.js         Загрузка каталога, работа чата, localStorage, API-запросы
```

Во frontend есть три вкладки:

- `Витрина` - карточки облачных сервисов.
- `Агент` - диалоговый подбор инфраструктуры.
- `О проекте` - краткое описание и ссылка на репозиторий.

Память чата хранится в `localStorage` браузера. Backend не хранит историю сообщений между пользователями. Кнопка `Сбросить чат` очищает локальную историю и начинает новый `chat_id`.

Поведение поля ввода:

- `Enter` отправляет сообщение;
- `Shift + Enter` оставляет перенос строки;
- кнопка сброса очищает историю текущего браузера.

Адрес API выбирается автоматически:

- если сайт открыт локально или как файл, API = `http://127.0.0.1:8000`;
- если сайт открыт с сервера, запросы идут на тот же домен через `/api/...`.
